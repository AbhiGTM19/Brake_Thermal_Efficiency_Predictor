About This Project

This web application is designed to predict the Brake Thermal Efficiency (BTE) of Internal Combustion Engines using a machine learning model. BTE is a crucial parameter indicating how efficiently an engine converts the chemical energy of fuel into useful work. Accurate prediction helps in optimizing engine design, fuel blend, and operational parameters.

Developed as part of an MLOps coursework, this project demonstrates a full-stack deployment of an ML model, from training and persistence to a user-friendly web interface.

The Prediction Model

The core of this application is a machine learning model trained on a comprehensive dataset of engine performance parameters. The model used is **Ridge Regression**, known for its effectiveness in handling multicollinearity and providing robust predictions.

Training Process:

  • Input features are scaled using `StandardScaler` to normalize their ranges and prevent bias towards features with larger values.
  • `GridSearchCV` is employed to find the optimal `alpha` parameter for the Ridge model, ensuring the best possible predictive performance.
  • The trained model and the scaler are saved using `joblib` into the `models/` directory, allowing them to be loaded efficiently by the Flask application for predictions.

The model considers key engine parameters such as Engine Load, Fuel Blend Percentage, Nanoparticle Concentration, Injection Pressure, and Engine Speed to make accurate BTE forecasts.
